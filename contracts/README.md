# Contracts 
## Install dependencies 

```bash
yarn install
```

## Run the linting and formatting checks on the contracts

```bash
yarn lint
```

## Run the tests

```bash
yarn test
```

## Build the project

```bash
yarn build
```

## Deploy to configured testnet

```bash
yarn deploy:dev
```

## Deploy to configured production network

```bash
yarn deploy
```

## Verify the contract on Etherscan

```bash
yarn verify
```
